ОПИСАНИЕ ФУНКЦИЙ В БИБЛИОТЕКЕ:

int ft_atoi(char *str) - преобразует число в строке в переменную типа int.

int ft_factorial (int b) - считает факториал числа.

int ft_power(int nb, int powel) - возводит число nb в степень power.

void ft_putchar(char c) - выводит символ c в терминал.

void ft_putnbr(int nb) - выводит число nb в терминал.

int ft_sqrt(int nb) - считает целый корень числа, иначе возвращает 0.

char *ft_strcat(char *dest, char *str) - добавляет строку str в конец строки dest и 
 возвращает указатель на результат.

int ft_strcmp(char *s1, char *s2) - сравнивает строки s1 и s2. Если s1 > s2 возвращает 1,
 если s1 < s2 возвращает -1, возврашает 0, если они равны. 

char *ft_strcpy(char *dest, char *src) - копирует строку src в строку dest и
 возвращает полученную строку.

int ft_strlen(char *str) - возвращает длину строки str.

char *ft_strncat(char *dest, char *str, int n) - добавляет n символов из строки str в конец
 строки dest и возвращает указатель на результат.

int *ft_strncmp(char *dest, char *str, int n) -  сравнивает первые n символов строки s1 и s2.
Если s1 > s2 возвращает 1, если s1 < s2 возвращает -1, возврашает 0, если они равны.

char *ft_strncpy(char *dest, char *str, int n) - копирует n символов строки str в строку dest.

char *ft_strstr(char *str, char *to_find) - ищет первое вхождение строки to_find в строке str и
возращает указатель на вхождение.
